# NameService for HyperLiquid

This is a trivial implementation of the Hyperliquid Name Service.

NameService.sol is an ERC721 with the following functions:
```
    function issueName(string memory _name, address _owner) external 
    function setAddress(string memory _name, address _address) external
    function nameToId(string memory _name) external returns uint256
    function nameToAddress(string memory _name) external returns address
```

It has been deployed to: `0xaCFf80214fbfD2291B900908dB04906c752e2aFD`

## Setup
create a `.env` file and add a `PRIVKEY` variable, used for deploying the contracts.

```
npx hardhat compile
npx hardhat test
npx hardhat ignition deploy .\ignition\modules\NameService.js [--network hyperliquid]
```
